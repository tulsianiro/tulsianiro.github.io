Campfire LD46

Campfire is a 2D platformer where you carry a lamp through a forest. As your lamp dies out, you lose vision of your surroundings. Rush from campfire to campfire to keep your lamp alive as you make your way to the end.

Key Features:

  • Basic Level Editing Functionality/Tooling
  • 2D Sprite Sheet Animation State Machine
  • 2D Custom Lighting
  • Platforming Mechanics and Physics
  • Audio System with Pooled Voices

Game Engine in DX11

Game Engine written in DX11. Supports various common Engine features, such as Skeletal Animation, Post FX like Bloom and Gaussian Blur, and more.

Key Features:

  • Normal Mapping
  • Skeletal Animation
  • Job System
  • Material System
  • Phong Lighting
  • Post FX (Bloom, Gaussian Blur)
  • Custom Memory Allocators
  • SIMD Math Library
April 10, 2017

Heightmap Terrain Renderer

Renders Heightmaps (Grayscale and RGB) in OpenGL, then allows the user to manipulate how that Heightmap is viewed in various ways.

Key Features:

  • Heightmap Rendering (Grayscale and RGB)
  • Texturing of Heightmap (Dynamically Calculate UV)
  • Dynamic Smoothing of Heightmap
  • Dynamic Calculation of Heightmap Normals
  • Blinn-Phong Lighting
  • User-Controlled Transformations of Heightmap
