Roller Coaster Simulation

Roller Coaster Simulation in OpenGL using custom file format to specify Catmull-Rom spline segments.

Key Features:

  • Render Catmull-Rom Splines as Roller Coaster Track
  • Skybox Rendering
  • Blinn-Phong Lighting
  • Calculate Frenet-Frame of Track Dynamically
  • Custom Trackfile
  • Chain Multiple Tracks Together With C1 Continuity
